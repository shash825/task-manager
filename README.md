# Task Manager

A JavaScript-based task management app for creating, tracking, and completing tasks.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000` in your browser

## What I Learned
Building this project helped me get more comfortable structuring a JavaScript application
from scratch. I practiced DOM manipulation, event handling, and keeping my code organized
across multiple functions.
